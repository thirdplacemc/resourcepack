# Resource Pack

A resource pack repository for Third Place MC.

## Overview

This repository contains custom textures, sounds, and other resources for the Third Place Minecraft server.

## Contents

- **textures/** - Custom block and item textures
- **sounds/** - Custom sound files
- **models/** - Custom block and item models
- **fonts/** - Custom font resources

## Installation

1. Download the latest resource pack from the releases
2. Place the folder in your `.minecraft/resourcepacks/` directory
3. Open Minecraft and go to Options → Resource Packs
4. Select this resource pack and move it to the active list
5. Click "Done" and restart the game
